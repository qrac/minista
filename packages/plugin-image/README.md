# minista-plugin-image

## About

[minista](https://minista.qranoko.jp) および [Vite](https://ja.vitejs.dev/) で動作するプラグイン。

## How To Use

```sh
$ npm install minista-plugin-image
```

```js
// ./minista.config.js
import { pluginImage } from "minista-plugin-image"

export default {
  plugins: [pluginImage()],
}
```
